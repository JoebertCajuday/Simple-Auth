## Simple Authentication System

<h3>A simple username/email and password authentication system that returns a jwt token on every succesful login. Built with Typescript</h3>


## Libraries used
<ul>

    <li>ExpressJs</li>
    <li>Sequelize</li>
    <li>Bcrypt</li>
    <li>JsonWebToken</li>
    <li>Body-Parser</li>
    <li>Morgan</li>
    <li>Helmet</li>
    <li>Cors</li>

</ul>
